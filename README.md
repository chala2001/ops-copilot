# Ops-Copilot

An AI-driven automation system for DevOps and SRE tasks.

## Project Structure

- `app.py`: Streamlit chat interface
- `rag.py`: RAG query engine
- `ingest.py`: Document ingestion pipeline
- `config.py`: Shared settings and constants
- `data/`: Raw documents (Markdown, Confluence, PDF)
- `chroma_db/`: Vector database storage
- `tests/`: Unit tests
